# =======================================================
# TROUBLESHOOTING
# =======================================================
#
# Error: TensorFlow installation fails
# Solution: Make sure Python 3.9-3.11, NOT 3.12+
#   python --version  (check version)
#
# Error: MediaPipe fails on Mac M1/M2
# Solution: Install special build
#   pip install mediapipe-silicon
#
# Error: OpenCV camera not working
# Solution: Install contrib version
#   pip install opencv-contrib-python
#
# Error: Keras model loading fails
# Solution: Install tf-keras
#   pip install tf-keras
