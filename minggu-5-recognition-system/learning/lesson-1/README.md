# Lesson 1: Build Complete Recognition Pipeline

## Tujuan
Build end-to-end recognition system dari dataset

## Pipeline Steps
1. Load dataset
2. Generate encodings
3. Save encodings (pickle)
4. Test recognition
5. Performance metrics

## Langkah
1. Copy dataset dari Minggu 4
2. Run: `python main.py`
3. System akan load & encode all faces
4. Test dengan webcam

## Metrics
- Accuracy
- Speed (FPS)
- False positives
