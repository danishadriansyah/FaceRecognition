üì¶ Project Module: face_recognizer.py

Core recognition engine yang menggunakan face_recognition library untuk identify people dari face encodings.

Class & Methods:

  • class FaceRecognizer(__init__) Initialize dengan tolerance level (default: 0.6) dan known faces database
  • encode_face(image, face_location) Generate 128-D encoding dari face region
  • recognize_face(unknown_encoding) Compare dengan known faces database, return best match dengan confidence
  • add_known_face(name, encoding) Add new person ke known faces database
  • load_known_faces(directory) Load semua known face encodings dari folder
  • calculate_confidence(distance) Convert face distance to confidence percentage (0-100%)

üí° Key Concepts

üî¢ Face Encodings (128-D Vector)

Setiap wajah di-convert menjadi 128-dimensional numeric vector yang unique:

import face_recognition encodings = face_recognition.face_encodings(image) # Output: array of 128 float numbers

üìè Face Distance & Comparison

Lower distance = more similar. Typical threshold = 0.6:

distance = face_recognition.face_distance(known_encodings, unknown_encoding) matches = face_recognition.compare_faces(known_encodings, unknown_encoding, tolerance=0.6)

üë§ Known Faces Database

Store known encodings dengan labels:

known_faces = { "John": [encoding1, encoding2], "Jane": [encoding3, encoding4] }

üéØ Tolerance Level

  • 0.4: Very strict (few false positives)
  • 0.6: Default (balanced)
  • 0.8: More lenient (may have false positives)
