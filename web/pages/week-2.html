üí° Key Concepts

üß† Haar Cascade

Machine Learning based object detection yang menggunakan cascade function trained dari ribuan positive dan negative images.

cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')

üîß detectMultiScale Parameters

  • scaleFactor: 1.1-1.5 (smaller = more accurate, slower)
  • minNeighbors: 3-6 (higher = fewer false positives)
  • minSize: (30,30) minimum face size in pixels

üé® ROI (Region of Interest)

Extract detected face area untuk further processing:

for (x, y, w, h) in faces: face_roi = image[y:y+h, x:x+w]

‚ö° Grayscale Conversion

Detection lebih cepat di grayscale karena single color channel:

gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY) faces = cascade.detectMultiScale(gray, 1.1, 5)
