üì¶ Project Module: recognition_service.py

High-level service yang orchestrate semua modules dengan optimized pipeline untuk real-time face recognition.

Class & Methods:

  • class RecognitionService(__init__) Initialize detector, recognizer, dataset manager dengan config options
  • load_dataset(dataset_path) Load dan cache semua known face encodings untuk fast lookup
  • recognize_from_image(image_path) Full pipeline: detect ‚Üí encode ‚Üí recognize untuk single image
  • recognize_from_webcam(camera_id=0) Real-time recognition dengan optimized frame processing
  • process_frame(frame, skip_frames=3) Process video frame dengan frame skipping untuk performance
  • get_recognition_stats() Return performance metrics: FPS, total recognized, accuracy rate
  • update_dataset() Hot reload dataset tanpa restart service
üìå Integration dengan Week 1-4:

Service ini uses face_detector.py (Week 2), face_recognizer.py (Week 3), dataset_manager.py (Week 4), dan image_utils.py (Week 1).

üèóÔ∏è Service Architecture

Recognition pipeline flow:

Input (Image/Frame) ‚Üí FaceDetector.detect_faces() ‚Üí FaceRecognizer.encode_face() ‚Üí FaceRecognizer.recognize_face() ‚Üí Return: [(name, confidence, bbox)]

‚ö° Optimization Techniques

  • Frame Skipping: Process every Nth frame (3-5)
  • Encoding Cache: Load encodings once at startup
  • Multi-threading: Separate threads untuk capture & process
  • Resize Frames: Scale down untuk faster detection (640x480)
