üëÅÔ∏è Lesson 2.1: Face Detection Basics

Understand Haar Cascade, load classifier, detect faces in static images

üéØ Learning Objectives

  • Understand Haar Cascade algorithm untuk face detection
  • Load pre-trained classifier dari OpenCV
  • Detect faces dalam static images
  • Draw bounding boxes around detected faces

ÔøΩ Tutorial

Step 1: Load Haar Cascade Classifier

import cv2

# Load pre-trained face cascade
face_cascade = cv2.CascadeClassifier(
    cv2.data.haarcascades + 'haarcascade_frontalface_default.xml'
)

# Verify cascade loaded
if face_cascade.empty():
    print("Error: Could not load cascade")
else:
    print("Face cascade loaded successfully!")

Haar Cascade classifier adalah pre-trained model yang bisa detect faces, eyes, dan objects lainnya.

Step 2: Load & Prepare Image

# Load image
img = cv2.imread('test_face.jpg')

# Convert to grayscale (required for cascade)
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

Haar Cascade bekerja pada grayscale images untuk better performance.

Step 3: Detect Faces

# Detect faces
faces = face_cascade.detectMultiScale(
    gray,
    scaleFactor=1.1,    # Image pyramid scale
    minNeighbors=5,     # Minimum neighbors for detection
    minSize=(30, 30)    # Minimum face size
)

print(f"Found {len(faces)} faces!")

detectMultiScale() returns list of rectangles (x, y, width, height) untuk setiap face detected.

Step 4: Draw Rectangles

# Draw rectangles around faces
for (x, y, w, h) in faces:
    cv2.rectangle(img, (x, y), (x+w, y+h), (0, 255, 0), 2)
    cv2.putText(img, 'Face', (x, y-10), 
                cv2.FONT_HERSHEY_SIMPLEX, 0.9, (0, 255, 0), 2)

# Display result
cv2.imshow('Face Detection', img)
cv2.waitKey(0)
cv2.destroyAllWindows()
