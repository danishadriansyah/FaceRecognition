üß¨ Lesson 3.1: Face Encodings Basics

Install face_recognition library, understand 128-dimensional vectors, generate face encodings

üéØ Learning Objectives

  • Install face_recognition library
  • Understand 128-dimensional face encoding vectors
  • Generate face encodings dari detected faces
  • Save & load encodings menggunakan pickle

üìù Tutorial

Step 1: Install face_recognition

pip install face_recognition

Library ini built on top of dlib, jadi akan install dlib juga sebagai dependency.

Step 2: Generate Face Encoding

import face_recognition
import cv2

# Load image
image = face_recognition.load_image_file('person.jpg')

# Get face encoding (128-D vector)
encodings = face_recognition.face_encodings(image)

if len(encodings) > 0:
    face_encoding = encodings[0]
    print(f"Encoding shape: {face_encoding.shape}")
    print(f"First 5 values: {face_encoding[:5]}")

Face encoding adalah array of 128 numbers yang uniquely represent seseorang's face.

Step 3: Save Encoding dengan Pickle

import pickle

# Save encoding
data = {
    'name': 'John Doe',
    'encoding': face_encoding
}

with open('john_encoding.pkl', 'wb') as f:
    pickle.dump(data, f)

print("Encoding saved!")

Step 4: Load Encoding

# Load encoding
with open('john_encoding.pkl', 'rb') as f:
    loaded_data = pickle.load(f)

print(f"Loaded name: {loaded_data['name']}")
print(f"Encoding loaded successfully!")
