üëã Lesson 1.1: Hello OpenCV

First steps dengan OpenCV: instalasi, import library, dan display image pertama kamu!

üéØ Learning Objectives

  • Install OpenCV menggunakan pip
  • Import cv2 library dan check version
  • Load & display image menggunakan cv2.imshow()
  • Understand waitKey() dan destroyAllWindows()

ÔøΩ Tutorial

Step 1: Install OpenCV

Buka terminal/command prompt dan install opencv-python:

pip install opencv-python numpy

Step 2: Import Library

Create file baru 01_hello_opencv.py dan import OpenCV:

import cv2
import numpy as np
print("OpenCV version:", cv2.__version__)

Step 3: Display Image

Load dan display image pertama kamu:

import cv2

# Load image
img = cv2.imread('test_image.jpg')

# Display image
cv2.imshow('My First Image', img)

# Wait for keypress
cv2.waitKey(0)

# Close all windows
cv2.destroyAllWindows()

Window akan muncul menampilkan image. Press any key untuk close window.

‚úÖ Exercises

  1. Find an image file (JPG/PNG) atau download dari internet
  2. Load image tersebut menggunakan cv2.imread()
  3. Display image dengan cv2.imshow()
  4. Print image dimensions: print(img.shape)
