‚úèÔ∏è Lesson 1.3: Drawing Shapes & Text

Draw rectangles, circles, lines, polygons, dan add text annotations pada images

üéØ Learning Objectives

  • Draw rectangles untuk bounding boxes
  • Draw circles dan ellipses
  • Draw lines dan polylines
  • Add text annotations dengan custom fonts
  • Understand color formats (BGR tuples)

üìù Tutorial

Step 1: Draw Rectangles

import cv2
import numpy as np

# Create blank image
img = np.zeros((480, 640, 3), dtype=np.uint8)

# Draw rectangle (filled)
cv2.rectangle(img, (100, 100), (300, 300), (0, 255, 0), -1)

# Draw rectangle (outline only)
cv2.rectangle(img, (350, 150), (550, 350), (255, 0, 0), 3)

Step 2: Draw Circles

# Draw circle (filled)
cv2.circle(img, (320, 240), 100, (0, 0, 255), -1)

# Draw circle (outline)
cv2.circle(img, (320, 240), 150, (255, 255, 0), 5)

Step 3: Draw Lines

# Draw single line
cv2.line(img, (0, 0), (640, 480), (255, 255, 255), 2)

# Draw polyline (connected points)
points = np.array([[100, 100], [200, 50], [300, 100], 
                  [400, 50], [500, 100]], np.int32)
cv2.polylines(img, [points], False, (0, 255, 255), 3)

Step 4: Add Text

# Add text to image
font = cv2.FONT_HERSHEY_SIMPLEX
cv2.putText(img, 'Hello OpenCV!', (50, 50), 
            font, 1, (255, 255, 255), 2)

# Multi-line text
cv2.putText(img, 'Line 1', (50, 100), font, 0.7, (0, 255, 0), 2)
cv2.putText(img, 'Line 2', (50, 150), font, 0.7, (0, 255, 0), 2)
