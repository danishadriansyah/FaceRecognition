üñºÔ∏è Lesson 1.2: Image Operations

Learn image manipulation: read, write, resize, crop, rotate, flip, dan color space conversion

üéØ Learning Objectives

  • Read dan write images menggunakan cv2.imread() dan cv2.imwrite()
  • Resize images dengan cv2.resize() dan berbagai interpolation methods
  • Crop images menggunakan array slicing
  • Rotate dan flip images
  • Convert color spaces (BGR ‚Üî RGB ‚Üî Grayscale ‚Üî HSV)

ÔøΩ Tutorial

Step 1: Read & Write Images

Load image, modify, dan save ke file baru:

import cv2

# Read image
img = cv2.imread('input.jpg')

# Write/save image
cv2.imwrite('output.jpg', img)

# Read as grayscale directly
gray = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)

Step 2: Resize Images

Resize image ke dimensions tertentu:

# Resize to specific width & height
resized = cv2.resize(img, (640, 480))

# Resize by scale factor (50%)
small = cv2.resize(img, None, fx=0.5, fy=0.5)

# Resize with interpolation
resized_smooth = cv2.resize(img, (800, 600), 
                            interpolation=cv2.INTER_LINEAR)

Step 3: Crop Images

Crop region menggunakan array slicing:

# Crop region: [y1:y2, x1:x2]
cropped = img[100:400, 200:500]

# Get image dimensions
height, width = img.shape[:2]

# Crop center region
center_crop = img[height//4:3*height//4, 
                  width//4:3*width//4]

Step 4: Rotate & Flip

Transform images dengan rotation dan flipping:

# Rotate 90 degrees clockwise
rotated = cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)

# Flip horizontal
flipped_h = cv2.flip(img, 1)

# Flip vertical
flipped_v = cv2.flip(img, 0)

# Flip both axes
flipped_both = cv2.flip(img, -1)

Step 5: Color Space Conversion

Convert between different color spaces:

# BGR to RGB
rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)

# BGR to Grayscale
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# BGR to HSV
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

# Grayscale to BGR (3-channel)
bgr_from_gray = cv2.cvtColor(gray, cv2.COLOR_GRAY2BGR)
