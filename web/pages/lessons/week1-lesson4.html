üìπ Lesson 1.4: Webcam Basics

Access webcam, capture frames, display real-time video feed

üéØ Learning Objectives

  • Access webcam menggunakan cv2.VideoCapture()
  • Read frames dalam continuous loop
  • Display real-time video stream
  • Save video recordings ke file

üìù Tutorial

Step 1: Access & Display Webcam

import cv2

# Open webcam (0 = default camera)
cap = cv2.VideoCapture(0)

while True:
    ret, frame = cap.read()
    if not ret:
        break
    
    cv2.imshow('Webcam Feed', frame)
    
    # Press 'q' to quit
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()

Step 2: Check Webcam Properties

# Get camera properties
width = int(cap.get(cv2.CAP_PROP_FRAME_WIDTH))
height = int(cap.get(cv2.CAP_PROP_FRAME_HEIGHT))
fps = int(cap.get(cv2.CAP_PROP_FPS))

print(f"Resolution: {width}x{height}")
print(f"FPS: {fps}")

Step 3: Save Video Recording

# Define codec & create VideoWriter
fourcc = cv2.VideoWriter_fourcc(*'XVID')
out = cv2.VideoWriter('output.avi', fourcc, 20.0, (640, 480))

while True:
    ret, frame = cap.read()
    if ret:
        out.write(frame)
        cv2.imshow('Recording', frame)
    
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

out.release()
