‚öôÔ∏è Lesson 2.3: Parameter Tuning & Optimization

Deep dive into scaleFactor, minNeighbors, minSize parameters

üéØ Learning Objectives

  • Understand scaleFactor (1.1-1.3) untuk image pyramid
  • Configure minNeighbors (3-6) untuk reduce false positives
  • Set minSize/maxSize untuk filter detection results
  • Balance accuracy vs speed

üìù Tutorial

scaleFactor Parameter:

# scaleFactor = 1.1 (slow, accurate)
faces = cascade.detectMultiScale(gray, scaleFactor=1.1, minNeighbors=5)

# scaleFactor = 1.3 (fast, less accurate)
faces = cascade.detectMultiScale(gray, scaleFactor=1.3, minNeighbors=5)

minNeighbors Parameter:

# minNeighbors = 3 (many detections, more false positives)
faces = cascade.detectMultiScale(gray, 1.1, minNeighbors=3)

# minNeighbors = 6 (fewer detections, less false positives)
faces = cascade.detectMultiScale(gray, 1.1, minNeighbors=6)

minSize/maxSize:

# Only detect faces >= 50x50 pixels
faces = cascade.detectMultiScale(gray, 1.1, 5, minSize=(50, 50))

# Only detect faces between 50x50 and 300x300
faces = cascade.detectMultiScale(gray, 1.1, 5, 
                                minSize=(50, 50), 
                                maxSize=(300, 300))

  1. Test different scaleFactor values (1.1, 1.2, 1.3)
  2. Experiment dengan minNeighbors (3, 4, 5, 6)
  3. Find optimal parameters untuk your webcam
