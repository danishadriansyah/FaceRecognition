# Sample Images

Place your test images here for testing face detection and recognition.

## Recommended Images:

1. **Individual portraits** - Single person, frontal face
2. **Group photos** - Multiple people
3. **Various lighting** - Indoor, outdoor, bright, dim
4. **Different angles** - Frontal, slight angle

## Naming Convention:

```
test_images/
├── single_person_1.jpg
├── single_person_2.jpg
├── group_photo_1.jpg
├── group_photo_2.jpg
└── various_angles_1.jpg
```

## Tips:

- Use high-quality images (not blurry)
- Good lighting
- Clear faces
- JPEG or PNG format
